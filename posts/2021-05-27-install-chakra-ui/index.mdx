## まえがき

Chakra-UI を自分のブログに導入したので、経緯を記載するよ。

## インストール

以下コマンドでプラグインをインストールする。
（プラグインの依存で Chakra-UI をインストールするようになっている）

```bash:bash
$ npm install @chakra-ui/gatsby-plugin
```

## gatsby-config.js 設定

`gatsby-config.js`の設定は以下の通り。  
ブラウザ毎の style 差分をなくすため、resetCSS は有効にする。

```js:gatsby-config.js {9,10}
module.exports = {
  // ...

  plugins: [
    // ...

    {
      resolve: "@chakra-ui/gatsby-plugin",
      options: {
        resetCSS: true,
      },
    },
  ],
}
```

まあほぼ [こ ↑ こ ↓](https://chakra-ui.com/docs/getting-started) に載ってる。
